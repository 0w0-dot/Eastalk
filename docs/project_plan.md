## 배포 계획 (Render)
### 📋 배포 개요
- **플랫폼**: Render.com (무료 시작 가능)
- **데이터베이스**: MongoDB Atlas (무료 512MB)
- **예상 시간**: 30분
- **비용**: 무료 → $7/월 (필요시 업그레이드)

### 🎯 배포 단계
1. **GitHub 저장소 준비** ✅ 완료
2. **MongoDB Atlas 설정** (필요시)
3. **Render 배포**
4. **환경변수 설정**
5. **배포 확인 및 테스트**

### 🔐 필요한 환경변수
```
MONGODB_URI=mongodb://localhost:27017/eastalk (또는 MongoDB Atlas URL)
NODE_ENV=production
PORT=3000
```
